# Project Title

Graph-Algorithms is an interactive website to draw graphs and visualize the shortest-path solutions calculated by 6 algorithms.

## Featured Algorithms

- Breadth First Search
- Depth First Search
- A\* Star
- Dijkstra's Algorithm
- Greedy (naive)
- Greedy (smart)

## Built With

- Vanilla JavaScript
- HTML, CSS
